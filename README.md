# 🚀 Space Shooter

A classic 2D arcade game inspired by Space Invaders, built in **C++** using the **Allegro graphics library**.

> 1st Semester project — BSCS, Sukkur IBA University (Fall 2023)  
> **Team Alpha:** Sohail Akhtar, Rattan Kumar, Hamza Butt

---

## 🎮 Gameplay

Pilot your spaceship through waves of enemies. Shoot them down, dodge collisions, and survive as long as possible to rack up the highest score.

**Controls:**
| Key | Action |
|-----|--------|
| ↑ ↓ ← → | Move spaceship |
| `Spacebar` | Fire bullet |

---

## ⚙️ Features

- Smooth **60 FPS** gameplay
- Enemy wave spawning with collision detection
- Lives system & real-time score tracking
- 800×400 game window

---

## 🛠️ Tech Stack

- **Language:** C++
- **Graphics Library:** Allegro 5

---

## 🚀 Getting Started

1. Install the Allegro library — download from the Google Drive link below
2. Clone this repo
3. Compile and run
